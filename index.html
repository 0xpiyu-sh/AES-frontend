Operations Guide

🔐 How to Use – Encryption

  1. Enter the plaintext message you want to secure.
  2. Provide a keyword (encryption key).
  3. Click on Initialize Encryption.
  4. The system generates a unique ciphertext using dynamic shifts.
  5. Copy and share the encrypted output securely.

🔓 How to Use – Decryption

  1. Paste the ciphertext you received.
  2. Enter the exact same keyword used during encryption.
  3. Click on Execute Decryption.
  4. The system reconstructs the key dynamically.
  5. Original message is retrieved safely.
